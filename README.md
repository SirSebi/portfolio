# Sebastian Brandes Portfolio

This is my personal portfolio website, developed with [Next.js](https://nextjs.org) and modern web technologies.

![Memoji Favicon](/public/default-memoji.png)

## Features

- Responsive design for all devices
- Dark color scheme
- Memoji as favicon
- Progressive Web App (PWA) support
- Optimized for search engines (SEO)

## Technologies

- [Next.js](https://nextjs.org) - React Framework
- [TypeScript](https://www.typescriptlang.org/) - Type-safe JavaScript
- [Tailwind CSS](https://tailwindcss.com) - Utility-First CSS Framework
- [Geist Font](https://vercel.com/font) - Modern font by Vercel

## Development

To start the development environment:

```bash
# Install dependencies
npm install
# or
pnpm install
# or
yarn install

# Start the development server
npm run dev
# or
pnpm dev
# or
yarn dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to see the result.

## Deployment

The website is optimized for deployment on [Vercel](https://vercel.com), but can also be deployed on other hosting platforms.

```bash
# Build for production
npm run build
# or
pnpm build
# or
yarn build
```
